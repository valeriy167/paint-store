# Paint Store — Интернет-магазин лакокрасочных материалов

**Зачётный проект по веб-разработке**  
Студент: ИУ6-33Б  
Технологии:  
- Frontend: React + JavaScript + Ant Design  
- Backend: Django + DRF (Django REST Framework)  
- База данных: PostgreSQL / SQLite (MVP)  
- Размещение: GitHub Pages / Vercel + Render / Heroku  

## Описание
Сайт-магазин лакокрасочных изделий с функционалом:
- Просмотр товаров в виде карточек
- Корзина с подсчётом стоимости
- Отзывы с модерацией
- Контакты с Яндекс.Картой
- Авторизация (необязательная)
- Административная панель (CRUD товаров, модерация отзывов)
